# Loan Approval Prediction

## Objective
Build a machine learning model to predict loan approval using borrower information.

## Dataset
Loan Approval Prediction Dataset from Kaggle.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- SMOTE
- Matplotlib

## Models
1. Logistic Regression
2. Random Forest

## Results

| Model | Precision | Recall | F1 Score | ROC-AUC |
|---------|---------|---------|---------|---------|
| Logistic Regression | 0.849 | 0.929 | 0.888 | 0.833 |
| Random Forest | 0.843 | 0.882 | 0.862 | 0.772 |

## Best Model
Logistic Regression with SMOTE

## Business Impact
The model helps financial institutions automate loan approval decisions and reduce manual screening efforts.
